# Installation on Local Machine

$ git clone https://github.com/ayushi-bhardwaj/react-dashboard.git

#Change the directory 
$ cd "react-dashboard"

#Install all the required dependencies
$ npm install

#Start the application on localhost:3000
@ npm start
